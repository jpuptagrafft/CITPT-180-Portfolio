Creating & Modifying a Video Game Engine

Within my Game Engine Design Class, we created a basic Game Engine in C and C++, which allowed for the User to jump infinitely, shoot balls indefinitely, have collision, and basic, mainly stationary AI.
For the Final of that class, I modified the system to be more in-line with what you would expect for a First Person Shooter: Only one jump, Multiple weapons, an Ammo system, more variety of AI, and some fake walls for secret areas.

Motivation with Motivate Me! An App made with Ionic

Can you see what Eye see? Tracing Axons through the layers of a Mouse’s Retina

During our Image Processing Class, we were asked to create an extension of an existing app (Named CASTER) that could trace an Axon through the microscope slices of a Mouse Retnia.
The tracing process was done with code written in Processing3, which is an IDE, and the tracing code used modifications so it could use C++-esque code, which made it an excellent base to start from. Processing3's functionality itself was very nice, as it meant that I didn’t have to load the entire slice below the previous slice for the tracing process; I just needed to load certain, much smaller segments, which made it rather convenient, as each slice was around 9MB in size.
